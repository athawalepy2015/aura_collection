# Aura Collection 🎨

Aura Collection is a creative digital art portfolio website built using React and Vite. The website showcases anime art, pencil sketches, colored artwork, and creative illustrations in a modern responsive gallery.

## Features

- Dark Mode
- Artwork Categories
- Interactive Image Modal
- Cloudinary Image Hosting
- Artwork Descriptions
- Responsive Design
- Image Protection
- Watermark Support

## Technologies Used

- React
- Vite
- CSS
- Cloudinary
- Lucide React Icons

## Categories

- Anime Art
- Pencil Sketches
- Colored Pencil Art
- Paintings
- Nature Art
